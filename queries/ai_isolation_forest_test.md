# آزمون شناسایی ناهنجاری با Isolation Forest
# AI-Powered Anomaly Detection using Isolation Forest

## توضیحات
این آزمون از الگوریتم یادگیری ماشین Isolation Forest برای شناسایی خودکار تراکنش‌های منزوی و غیرعادی استفاده می‌کند.

### ویژگی‌های استفاده شده
- **مبلغ تراکنش**: مبلغ بدهکار یا بستانکار
- **ساعت روز**: ساعت انجام تراکنش (0-23)
- **روز هفته**: روز هفته انجام تراکنش (0-6)
- **کد حساب**: کد حساب مالی (هش شده)

### نحوه کار
1. تمام تراکنش‌ها را از پایگاه داده می‌خواند
2. ویژگی‌های عددی را استخراج و نرمال‌سازی می‌کند
3. مدل Isolation Forest را آموزش می‌دهد
4. تراکنش‌های منزوی را شناسایی و رتبه‌بندی می‌کند

### پارامترها
- **columnName**: نام ستون برای آزمون (Debit یا Credit)
- **contamination**: نسبت ناهنجاری مورد انتظار (0.1 = 10%)
- **minAmount**: حداقل مبلغ برای بررسی

### خروجی
تراکنش‌هایی که به عنوان ناهنجاری شناسایی شده‌اند با:
- امتیاز ناهنجاری (پایین‌تر = مشکوک‌تر)
- رتبه‌بندی بر اساس میزان انحراف
- اطلاعات کامل تراکنش

### کاربردها
- شناسایی تراکنش‌های مشکوک
- تشخیص الگوهای غیرعادی در زمان‌بندی
- یافتن تراکنش‌های خارج از الگوی معمول

### نیازمندی‌ها
- scikit-learn >= 1.3.0
- numpy >= 1.26.0
- حداقل 10 تراکنش برای آموزش مدل
